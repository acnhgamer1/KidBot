# 🤖 KidBot AI - ChatGPT-like AI Assistant

A powerful AI chat application with ChatGPT-like interface, featuring text conversations, image generation, and advanced "deep thinking" capabilities. Built with React and TypeScript, deployable to GitHub Pages.

## ✨ Features

- **💬 ChatGPT-like Interface** - Modern dark theme with message bubbles and typing indicators
- **🤖 AI Chat** - Intelligent conversations powered by Pollinations AI
- **🧠 Deep Think Mode** - Dual AI processing for enhanced, more thoughtful responses
- **🎨 Image Generation** - Create images from text descriptions using AI
- **📱 Responsive Design** - Works perfectly on desktop and mobile devices
- **🌐 GitHub Pages Ready** - Deploy for free with automatic CI/CD

## 🚀 Quick Start

### Option 1: Deploy to GitHub Pages (Recommended)

1. **Fork this repository** or use it as a template
2. **Enable GitHub Pages** in your repository settings
3. **Push to main branch** - automatic deployment will begin
4. **Access your app** at `https://yourusername.github.io/KidBot/`

📖 **Detailed deployment instructions:** [DEPLOY.md](DEPLOY.md)

### Option 2: Run Locally

```bash
# Clone the repository
git clone https://github.com/yourusername/KidBot.git
cd KidBot

# Install dependencies
cd client
npm install

# Start development server
npm run dev
```

Visit `http://localhost:5173/KidBot/` to use the app locally.

## 🎯 How to Use

1. **Normal Chat** - Ask questions, have conversations, get help with anything
2. **Deep Think Mode** - Toggle for complex questions that need extra analysis
3. **Image Generation** - Switch to image mode and describe what you want to see
4. **Clear Chat** - Start fresh conversations anytime

## 🛠️ Technology Stack

- **Frontend:** React 19, TypeScript, Vite
- **Styling:** Modern CSS with ChatGPT-inspired design
- **AI APIs:** Pollinations AI for text and image generation
- **Testing:** Vitest, React Testing Library
- **Deployment:** GitHub Pages with GitHub Actions
- **Development:** Hot reloading, ESLint, Prettier
